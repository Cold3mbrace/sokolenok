# v51.3 — polish: анимации, hover, skeleton

Сайт ощущается живым и быстрым, без тяжёлых эффектов.

## Что добавилось

### 1. Hover-эффекты на карточках
- Посты в ленте при hover поднимаются на 2px, граница подсвечивается зелёным, появляется тень
- Заголовок поста подсвечивается зелёным
- Карточки превью лендинга, диалоги, уведомления — лёгкая зелёная подсветка фона при hover

### 2. Stagger-fade-in для списков
- Лента, уведомления при появлении делают плавный fade+slide-up с шагом 40ms между элементами
- До 12 элементов staggered, остальные fade'ятся вместе (без тормозов)
- Уважает `prefers-reduced-motion` (выключается у юзеров с anti-motion в системе)

### 3. Skeleton loaders для ленты
- Вместо "Загружаем ленту…" со спиннером — 3 шиммер-плейсхолдера в форме реальных постов
- Юзер сразу видит «как будто что-то загрузилось» — психологически быстрее

### 4. Микро-анимация лайков
- Сердечко при лайке делает быстрый pulse-scale (1 → 1.35 → 0.92 → 1)
- Число счётчика тоже скачет + кратко окрашивается зелёным
- При unlike — без анимации (не привлекаем к негативному действию)

### 5. Toast slide-in
- Тосты появляются с лёгким slide-from-right (24ms)

### Кнопка "наверх" на лендинге
- Старая статичная стрелка-вниз заменена на функциональную "back to top"
- Прячется когда юзер вверху, появляется при скролле >600px
- Плавающая кнопка справа снизу с hover-зелёным

## Проверка
- Зайди в ленту — должен быть skeleton сразу, потом stagger fade-in
- Полайкай пост — сердечко пульсирует
- Скрольни лендинг вниз — справа снизу появляется кнопка наверх
