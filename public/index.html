Безопасность

Только чтение. Никакого риска

Вход через Steam OpenID
Мы никогда не видим твой пароль. Авторизация происходит на стороне Steam — мы получаем только публичный SteamID.
Доступ только на чтение
Сервис не совершает никаких действий с аккаунтом: не торгует, не отправляет заявки, ничего не меняет.
Публичные данные
Используются только официальные открытые API Steam и Faceit. Если профиль закрыт — мы честно об этом скажем.

FAQ

Частые вопросы

Это безопасно? Вы не угоните аккаунт?

Вход идёт через официальный Steam OpenID — мы не получаем твой пароль и не можем совершать действия с аккаунтом. Доступ только на чтение публичных данных.

Нужен ли Faceit, чтобы пользоваться?

Нет. Базовая статистика и инвентарь работают только на данных Steam. Faceit добавляет детальную аналитику матчей, если у тебя есть привязанный аккаунт.

Почему статистика Steam «за всё время»?

Steam отдаёт только суммарные показатели за всю историю CS (CS:GO + CS2). Детальная статистика по конкретным матчам берётся из Faceit.

Можно проверить чужой аккаунт?

Да. Вставь SteamID или ссылку на профиль на странице проверки — увидишь публичную статистику, инвентарь, баны и автоматические флаги подозрительности.
