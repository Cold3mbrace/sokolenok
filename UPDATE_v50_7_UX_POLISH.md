# SOKOLENOK v50.7 — UX polish

## What changed

- Empty cards now show their real title instead of the generic "no data" line.
- Empty states can show multiple next actions.
- Public pages with no posts now guide owners to create a post, and visitors back to feed/communities.
- Messages empty states now offer direct actions: find friends or check a player.
- Message load failures now show a retry action instead of a dead empty panel.
- Notifications load failure now shows clear recovery actions.
- Communities empty tabs now guide users to create, browse all communities, or return to feed.
