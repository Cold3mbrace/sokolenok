# v50.14 — iPhone boot guard

Что изменилось:
- Добавлен `/boot-guard.js`, который запускается до основного приложения.
- Если `app.js` падает или не стартует на iPhone/Safari, вместо белого экрана появляется экран восстановления.
- Кнопка `Обновить` на экране восстановления сбрасывает service worker и перезагружает страницу.
- Версии CSS/JS в HTML обновлены, чтобы iPhone не держал старый кэш.

Проверка:
- `npm run check`
