# v50.17 — lighter start, PWA restored

## Что изменилось

- PWA-поведение возвращено как в `v50.15`: `manifest`, iOS standalone meta и service worker снова включены.
- Временное отключение PWA из `v50.16` удалено.
- Сервер теперь отдает HTML/CSS/JS/JSON/SVG/TXT с gzip, если браузер поддерживает сжатие.
- Фоновые действия после входа запускаются чуть позже, чтобы первый экран рисовался легче:
  - счетчик непрочитанных;
  - realtime/WebSocket;
  - heartbeat активности;
  - page_view аналитика.
- Все обычные страницы подключают `/styles.css?v=50.17` и `/app.js?v=50.17`.
