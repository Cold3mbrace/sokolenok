# SOKOLENOK v50.2 — First visit without questions

## What changed

- Added a first-visit action block on the logged-in main page:
  - check a player,
  - open my public profile,
  - find and add friends.
- Improved empty states with clear next actions:
  - no friends -> find a player,
  - no messages -> add a friend,
  - no notifications -> explains what will appear there and links to useful areas.
- Added a Telegram-only profile prompt to connect Steam for CS2 stats, inventory and full profile checks.
- Made social profile actions more prominent.
- Reworded the mobile friends menu from "ЧС" to "чёрный список" and softened "recommendations" into friend suggestions.
