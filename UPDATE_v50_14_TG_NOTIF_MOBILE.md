# v50.14 — Telegram names + mobile notification popover

Что изменилось:
- Уведомления больше не показывают сырой `tg:123456789`.
- Telegram-пользователи в уведомлениях берутся из локального профиля SOKOLENOK.
- Если имени нет, показывается `Telegram-пользователь`, а не внутренний id.
- Выпадающее меню уведомлений на телефоне стало `fixed` по экрану и больше не уезжает вбок.

База:
- Сборка сделана от `v50.13-404-rollback`, без экспериментального `boot-guard`.
